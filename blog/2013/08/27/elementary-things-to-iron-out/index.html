Elementary Things to Iron Out

Elementary OS

With the number of Ubuntu clones increasing day by day and now with them being stable and visually stunning, ElementaryOS stands out. Its sleek and the title and menu bars in pantheon have just the right amount of gloss that works for me.(To Mint lovers out there, too shiny for my tastes, sorry!) Though now that i have used it for more than 2 weeks,there are some problems that need to be ironed out.

  • Files

    The biggest issue i seem to be having is that Files seems to have some trouble loading some particular filesystems. This issue seems to revolve around USB disks and drives with low space. Mounting harddisks, both internal and external seems to be no issue.

  • nm-applet

    Another quirky thing is that nm-applet does not automatically connect or even search for networks when booted. An easy fix is to choose connect to a hidden network, when it begins searches as well if you give a bogus hidden network name.Still have to find a permanent fix for this though.

  • Terminal

    To other ruby, octopress users out there: If you have any issue in using rvm in pantheon-teminal Just add the .bash_profile file to your .bashrc (Copy the syntax for the .bash_aliases file). This issue is simply because rvm uses a login shell to manage profiles within different folders.This page has some great info regarding the same.
